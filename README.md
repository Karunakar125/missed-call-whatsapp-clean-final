Turn missed calls into instant WhatsApp conversations.

## 🚨 Problem

Businesses lose leads when calls go unanswered.

---

## ⚙️ Solution

This system detects missed calls in real-time and sends a WhatsApp message to the customer within seconds.

---

## 🔄 Flow

1. Customer calls agent  
2. Call forwarded to virtual number (TeleCMI / Plivo)  
3. Webhook triggers FastAPI backend  
4. Missed call detected using event + duration  
5. WhatsApp message sent via Meta Cloud API  
6. Agent receives lead notification  

---

## 🧱 Tech Stack

- FastAPI (Python)
- TeleCMI / Plivo (webhooks)
- Meta WhatsApp Cloud API
- Event-driven architecture

---

## ⚡ Key Features

- Real-time missed call detection  
- Deduplication using CallUUID  
- Instant WhatsApp messaging  
- Agent notification system  

---
